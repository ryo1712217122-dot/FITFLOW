# FITFLOW - 筋トレ実績管理アプリ (Workout Tracker) - v1.9.2

ブラウザで動作する、美しく実用的なフィットネス実績・ヘルスケア管理アプリケーションです。
「記録する」タブは、ジムに行った日に入力する**有酸素＋筋トレフォーム**と、ジムに行かない日でも入力する**体重＋特別な飲食フォーム**の2つに分かれています。筋トレの種目は入力し終えるごとにその場で個別保存できるため、ジムでのリアルタイム入力にも使えます。Googleスプレッドシート（GASウェブアプリ）によるセキュアな双方向クラウド同期機能も備えています。

## 🎨 デザインテーマ
オーガニックな健康・ヘルスケアの親しみやすさを与える、自然なトーンの配色を基調としています。
- **ベースカラー (Slate-Teal)**: `#324851`
- **アクセントカラー (Active-Green)**: `#86AC41`
- **セカンダリカラー (Forest-Teal)**: `#34675C`
- **ミュートカラー (Sage-Teal)**: `#7DA3A1`

その他、設定タブから「ディープオーシャン」「サンセット」「アメジスト」の3つの追加カラーテーマにも切り替え可能です。

## 🚀 起動方法

### 方法1. ダブルクリックで開く (最も簡単)
`index.html` ファイルをブラウザにドラッグ＆ドロップするか、ダブルクリックして開きます。

### 方法2. ローカルサーバーを起動して開く (推奨)
ターミナルを開き、このフォルダ内で以下を実行します：
```bash
npm start
# または
npx serve .
```
起動後、ブラウザで指定のアドレス（通常は `http://localhost:3000`）にアクセスします。

## 🗂️ プロジェクト構成

ビルドステップなしで動くよう、CSS/JSともに素朴な `<link>`/`<script>` の読み込み順で依存関係を解決するクラシックスクリプト構成にしています（ESモジュールは使っていません。ダブルクリックで`file://`から開いても動くようにするためです）。

```
index.html            単一のHTML。5タブすべてのマークアップとscript/linkタグを持つ
css/
  layers.css           @layerの優先順位宣言（必ず最初に読み込む）
  tokens.css           CSS変数・カラーテーマの基礎トークン
  base.css              リセット・共通アニメーション
  layout.css            サイドバー・ヘッダー・カード・グリッドの骨格
  components.css        ボタン・フォーム部品・トースト・モーダルなど汎用UI
  pages/                タブごとの固有スタイル (record/dashboard/plan/history/settings)
  responsive.css        900px以下のレスポンシブ対応（最後に読み込む）
js/
  config.js              定数・設定値
  state.js                グローバル状態とlocalStorage永続化
  dom.js / utils.js / notifications.js / theme.js / navigation.js
  dashboard.js            ダッシュボードタブ（統計・カレンダー・グラフ・メンテナンスカロリー設定）
  record-form.js          記録するタブ（有酸素+筋トレフォーム・体重+食事フォーム。筋トレ種目は個別即時保存）
  history-workouts.js / history-logs.js / history-nav.js   履歴リストタブ
  plan.js                 最適化計画タブ + ダッシュボードの計画サマリーウィジェット
  sync.js                 Googleスプレッドシート(GAS)クラウド同期
  backup.js               テーマ配線・JSONエクスポート/インポート/初期化
  main.js                 エントリポイント（DOMContentLoaded初期化）
lib/data-utils.js       DOM非依存の純粋関数（`npm test`でテストされる唯一のファイル）
```

## 📱 モバイル優先 (Mobile-First) 設計

スマートフォンなど横幅が狭いデバイス（900px以下）からアクセスした場合、UIがモバイル表示に自動で最適化されます。
- **ピンズーム・拡大禁止＆スクロールロック**: モバイル操作時の不要な拡大縮小（ピンチズーム）や横揺れ（横スクロール）を防止し、完全に縦スクロールとタップのみで軽快に操作可能です。
- **ボトムナビゲーションバー**: 画面最下部にアプリ同様のタブ切り替えバーが固定され、片手で親指から直感的に操作できます。
- **レイアウトの縦スタック化**: 各種入力フォームやチャートが画面幅いっぱいに収まり、スクロールしやすい縦並び表示に変わります。
- **ヘッダーテーマ切り替え**: モバイルではヘッダー右上からワンタップでライト/ダークテーマを切り替えられます。

## 🔥 主な機能

1. **入力の目的で分かれた2つの記録フォーム**
   - **有酸素・筋トレの記録**（ジムに行った日に入力）: 日付・時刻・有酸素の走行距離・筋トレ種目をまとめて入力します。筋トレ種目は1種目入力し終えるごとに「この種目を保存」ボタンでその場に個別保存でき、ジムでのリアルタイム入力を想定しています。保存済みの種目はカードの枠線が色付きになり、ひと目で区別できます。
   - **体重・食事の記録**（ジムに行かない日でも入力）: 体重と特別な飲食をまとめて入力します。どちらも任意項目で、片方だけの入力も可能です。
   - 筋トレ種目の詳細入力（セット数、レップ数）はアコーディオンパネルに収められており、開閉して表示・非表示を切り替えられます。

2. **Googleスプレッドシート連携（クラウド同期）**
   - 自分専用のスプレッドシートとGAS（Google Apps Script）を接続し、ワークアウト・体重・有酸素・特別な飲食・最適化計画を含む全データのバックアップと復元をボタン一つで実行できます。
   - オンライン復帰時や起動時にも自動で同期を行いますが、クラウド側のデータ件数がローカルより大きく減っている場合は誤操作防止のため自動上書きをスキップします。

3. **ジムに行った日時の詳細記録**
   - ワークアウトの記録時に「日付」に加えて「開始時刻 (HH:MM)」を入力できます。履歴一覧でも日時として綺麗にタイムスタンプが並びます。

4. **体重トラッカー (Weight Tracker)**
   - 体重 (kg) を日付別で記録できます。
   - ダッシュボードに最新の体重が表示され、過去10件の推移が折れ線グラフで視覚化されます。
   - 「履歴」タブの「体重履歴」から過去の記録を一覧で確認し、誤った値の修正や削除ができます。

5. **ランニング消費カロリー計算 (Cardio Tracker)**
   - 走った距離 (km) を記録できます。
   - 記録時に「最新の体重 × 走行距離」から簡易的な消費エネルギー (kcal) を自動で計算・保存します。
   - 今日の総消費カロリーや走った距離のサマリーを即座にダッシュボードに反映します。
   - **週間ランニング目標**: ダッシュボードに今週（日〜土）の走行距離と目標（デフォルト15km、最適化計画タブで変更可能）に対する達成度をプログレスバーで表示します。

6. **筋トレの消費カロリー概算**
   - 筋トレ種目を保存するたびに、そのセッションの合計セット数から消費エネルギー (kcal) を簡易的に見積もります（1セットあたりの目安kcal × 合計セット数）。
   - 記録フォームにその場のセッション合計を表示するほか、履歴カードにも表示されます。

7. **メンテナンスカロリー比較 (Calorie Balance)**
   - 自分の1日の維持カロリー（メンテナンスカロリー）を**ダッシュボード**タブから調整・保存できます（デフォルト: 2000 kcal）。カロリーバランスグラフのすぐ下に設定欄があるので、調整した結果をその場で確認できます。
   - ダッシュボードの「カロリーバランス」グラフと「本日の総消費」では、有酸素の消費カロリーに加えて筋トレの推定消費カロリーも合算して表示します。

8. **最適化計画 (Plan)**
   - 摂取・消費カロリー目標や体重減少ロードマップ、生活習慣ルールをカスタマイズして保存できます。

## 🔒 データについて

- 本アプリの全データはブラウザの **LocalStorage** に安全に保存され、外部サーバーに送信されることはありません（クラウド同期を設定した場合のみ、自分のGASウェブアプリURLへ送信されます）。
- データのバックアップや移行は、「データ・設定」タブから JSON 形式で容易にエクスポート/インポート可能です（ワークアウト・体重・有酸素・特別な飲食・最適化計画をすべて含みます）。
