# FITFLOW - 筋トレ実績管理アプリ (Workout Tracker)

ブラウザで軽快に動作する、美しく実用的な筋トレ実績管理アプリケーションです。
カレンダー、各種グラフ、推定1RM推移などを備え、モチベーション維持をサポートします。

## 🚀 起動方法

このアプリは完全にクライアントサイド（HTML / CSS / JavaScript）で動作するため、特別なサーバー構築を行わずに利用可能です。以下のいずれかの方法で起動できます。

### 方法1. ダブルクリックで開く (最も簡単)
`workout-tracker` フォルダ内の `index.html` ファイルをお使いのブラウザ (Chrome, Safari, Edge など) にドラッグ＆ドロップするか、ダブルクリックして開きます。

### 方法2. ローカルサーバーを起動して開く (推奨)
ターミナルを開き、このフォルダ内で簡易サーバーを起動してアクセスします。

**Node.js (npx) を使う場合:**
```bash
npx serve .
# または
npx http-server .
```
起動後、ターミナルに表示されるURL（例: `http://localhost:3000` など）にブラウザでアクセスします。

**Python を使う場合:**
```bash
python3 -m http.server 8000
```
起動後、ブラウザで `http://localhost:8000` にアクセスします。

## 🔥 主な機能

1. **ダッシュボード (Dashboard)**
   - **統計情報**: これまでの総ワークアウト数、今月の実施日数、総挙上ボリューム（セットごとの `重量 × レップ数` の合計）を表示。
   - **継続日数 (Streak)**: 前日または当日にワークアウトがある限り、連続で実施した日数をカウント。
   - **トレーニングカレンダー**: 実施した日にはマークがつき、過去のスケジュールが一目でわかります。クリックするとその日の履歴を検索します。
   - **部位別トレーニング比率**: 登録したワークアウトのカテゴリー分布をドーナツグラフで視覚化。
   - **週別頻度グラフ**: 過去6週間分の実施頻度を棒グラフで表示。

2. **ワークアウト記録 (Log Workout)**
   - **基本情報**: 日付、ワークアウト名（自動補完リストあり）、部位カテゴリーを選択。
   - **コンディション**: 今日の気分や体調を5つの絵文字（🔥、💪、😊、🥱、☠️）から選択。
   - **感想・メモ**: メモや気づきをフリー入力。
   - **実施した種目**: 複数の種目を追加し、セット数、挙上重量、レップ数を動的に入力・調整可能。

3. **履歴と分析 (History & Analytics)**
   - **キーワード検索**: ワークアウト名や種目名、感想のキーワードから過去のログを部分一致で検索。
   - **フィルター絞り込み**: 部位カテゴリーや今日の気分から瞬時に抽出。
   - **推定1RM推移グラフ**: 選択した特定の種目について、過去の最大挙上重量と推定1RM（`重量 × (1 + レップ数 / 30)`）の推移を折れ線グラフで表示。成長曲線が可視化されます。
   - **編集・削除機能**: 過去の記録の修正や削除が可能です。

4. **データ管理・設定 (Data & Settings)**
   - **エクスポート**: 記録した全ワークアウトデータを JSON 形式でローカルPCにダウンロード保存します。
   - **インポート**: 保存した JSON ファイルを選択し、既存のデータとマージして復元・移行できます。
   - **初期化**: ローカルに保存されているすべてのワークアウトデータを安全に消去します。

## 🔒 データについて

- このアプリに入力されたデータは、すべてお使いのブラウザの **LocalStorage** に安全に保存されます。
- 外部のサーバーやデータベースには一切送信されないため、プライバシーが完全に保護されます。
- ブラウザのキャッシュクリアや別のパソコン・スマホへの移行時には、**データ・設定**タブから「エクスポート」を行い、バックアップファイルを保管することをお勧めします。
