# FITFLOW - 筋トレ実績管理アプリ (Workout Tracker) - v1.7.0

ブラウザで動作する、美しく実用的なフィットネス実績・ヘルスケア管理アプリケーションです。
「記録（ジム・有酸素・特別な飲食）」を一つのフォームにまとめた記録優先のUIと、体重だけを単独ですばやく記録できる専用フォーム、Googleスプレッドシート（GASウェブアプリ）によるセキュアな双方向クラウド同期機能を備えています。

## 🎨 デザインテーマ
オーガニックな健康・ヘルスケアの親しみやすさを与える、自然なトーンの配色を基調としています。
- **ベースカラー (Slate-Teal)**: `#324851`
- **アクセントカラー (Active-Green)**: `#86AC41`
- **セカンダリカラー (Forest-Teal)**: `#34675C`
- **ミュートカラー (Sage-Teal)**: `#7DA3A1`

その他、設定タブから「ディープオーシャン」「サンセット」「アメジスト」の3つの追加カラーテーマにも切り替え可能です。

## 🚀 起動方法

### 方法1. ダブルクリックで開く (最も簡単)
`index.html` ファイルをブラウザにドラッグ＆ドロップするか、ダブルクリックして開きます。

### 方法2. ローカルサーバーを起動して開く (推奨)
ターミナルを開き、このフォルダ内で以下を実行します：
```bash
npm start
# または
npx serve .
```
起動後、ブラウザで指定のアドレス（通常は `http://localhost:3000`）にアクセスします。

## 📱 モバイル優先 (Mobile-First) 設計

スマートフォンなど横幅が狭いデバイス（900px以下）からアクセスした場合、UIがモバイル表示に自動で最適化されます。
- **ピンズーム・拡大禁止＆スクロールロック**: モバイル操作時の不要な拡大縮小（ピンチズーム）や横揺れ（横スクロール）を防止し、完全に縦スクロールとタップのみで軽快に操作可能です。
- **ボトムナビゲーションバー**: 画面最下部にアプリ同様のタブ切り替えバーが固定され、片手で親指から直感的に操作できます。
- **レイアウトの縦スタック化**: 各種入力フォームやチャートが画面幅いっぱいに収まり、スクロールしやすい縦並び表示に変わります。
- **ヘッダーテーマ切り替え**: モバイルではヘッダー右上からワンタップでライト/ダークテーマを切り替えられます。

## 🔥 主な機能

1. **記録優先の直感的UIレイアウト**
   - アプリ起動時の第一画面は「記録する」タブ。日付・時刻・有酸素・特別な飲食・筋トレ（種目詳細含む）を1つのフォームで一括記録できます。体重はサイドの専用フォームから日付を指定して単独で記録します。
   - 筋トレ種目の詳細入力（セット数、レップ数）はアコーディオンパネルに収められており、開閉して表示・非表示を切り替えられます。

2. **Googleスプレッドシート連携（クラウド同期）**
   - 自分専用のスプレッドシートとGAS（Google Apps Script）を接続し、ワークアウト・体重・有酸素・特別な飲食・最適化計画を含む全データのバックアップと復元をボタン一つで実行できます。
   - オンライン復帰時や起動時にも自動で同期を行いますが、クラウド側のデータ件数がローカルより大きく減っている場合は誤操作防止のため自動上書きをスキップします。

3. **ジムに行った日時の詳細記録**
   - ワークアウトの記録時に「日付」に加えて「開始時刻 (HH:MM)」を入力できます。履歴一覧でも日時として綺麗にタイムスタンプが並びます。

4. **体重トラッカー (Weight Tracker)**
   - 体重 (kg) を日付別で記録できます。
   - ダッシュボードに最新の体重が表示され、過去10件の推移が折れ線グラフで視覚化されます。
   - 「履歴」タブの「体重履歴」から過去の記録を一覧で確認し、誤った値の修正や削除ができます。

5. **ランニング消費カロリー計算 (Cardio Tracker)**
   - 走った距離 (km) を記録できます。
   - 記録時に「最新の体重 × 走行距離」から簡易的な消費エネルギー (kcal) を自動で計算・保存します。
   - 今日の総消費カロリーや走った距離のサマリーを即座にダッシュボードに反映します。

6. **メンテナンスカロリー比較 (Calorie Balance)**
   - 自分の1日の維持カロリー（メンテナンスカロリー）を**データ・設定**タブから調整・保存できます（デフォルト: 2000 kcal）。
   - ダッシュボードの「カロリーバランス」グラフで、直近7日間のランニングによる消費カロリーとメンテナンスカロリーの閾値を並べて比較・確認できます。

7. **最適化計画 (Plan)**
   - 摂取・消費カロリー目標や体重減少ロードマップ、生活習慣ルールをカスタマイズして保存できます。

---

## 🔒 データについて

- 本アプリの全データはブラウザの **LocalStorage** に安全に保存され、外部サーバーに送信されることはありません（クラウド同期を設定した場合のみ、自分のGASウェブアプリURLへ送信されます）。
- データのバックアップや移行は、「データ・設定」タブから JSON 形式で容易にエクスポート/インポート可能です（ワークアウト・体重・有酸素・特別な飲食・最適化計画をすべて含みます）。
